# Seaborn Data Visualization Project

## Project Overview
This project focuses on data visualization using Python's Seaborn and Matplotlib libraries.  
The goal is to explore real-world datasets and extract meaningful insights through graphical analysis.

We worked with:
- Exercise dataset (gym + heart rate study)
- Planets dataset (astronomical data)

---

## Objectives

- Visualize relationships between variables using Seaborn
- Understand distribution patterns in datasets
- Compare categorical effects on numerical data
- Build professional-quality visualizations
- Practice data cleaning and transformation techniques

---

## Datasets Used

### 1. Exercise_Data.csv
Contains:
- Diet type
- Exercise type
- Pulse measurements at different time intervals

Used for:
- Heatmap analysis
- Categorical boxplots

---

### 2. Planets Dataset (Seaborn built-in)
Contains:
- Planet mass
- Orbital period
- Distance
- Discovery method
- Year of discovery

Used for:
- Relational plots
- Distribution plots
- Categorical plots

---

## Visualizations Created

### Exercise Dataset
- Heatmap of average pulse rates by diet and exercise type
- Boxplot showing pulse distribution across categories

### Planets Dataset
- Scatter plot (distance vs orbital period)
- Line plot (mass trends over time)
- Histogram (orbital period distribution)
- KDE plot (mass density distribution)
- Boxplot (orbital period by discovery method)
- Bar plot (average mass by method)

---

## Key Insights

- Exercise type and diet both influence pulse rate.
- Heart rate varies significantly across different activity conditions.
- Planets with greater distance from their star tend to have longer orbital periods.
- Discovery methods influence the type of planets detected.
- Planet mass distribution is skewed, with most planets being lower mass.
